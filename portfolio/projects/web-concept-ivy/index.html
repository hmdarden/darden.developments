Website Concept:
Ivy Karlsgodt

Project Description / Goals:

  1. Research and design a website concept for a visual artist to showcase their work, background, and artistic impact
  2. Creat clean, engaging layout that balances strong imagery with accessible content suitable for a general audience

Creative Process and Strategy:

  1. Conducted thorough research and information gathering to understand the artist's work and target audience needs
  2. Developed wireframes to establish site structure and intuitive navigation flow
  3. Created styleboards to establish site structure and intuitive navigation flow
  4. Built an interactive, high-fidelity prototype in Figma to visualize the final user experience

Skills Used and Learned

  • User Research
  • Wireframing
  • Styleboarding
  • High-Fidelity Prototyping (Figma)
  • Visual Design

Design Assets

  • Wireframe
  • Styleboard
  • Final Prototype

Results:

  • Delivered cohesive website concept that effectively highlights the artist's protfolio and story
  • Provided prototype that balances visual appeal with user-friendly navigation
  • Created design assets that can be used for further development and client presentations
